# React To-Do List App

This is a simple To-Do List application built with React.  
The app allows users to add, complete, and delete tasks.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Simple and clean interface

## Technologies Used

- React
- JavaScript
- HTML
- CSS

## Installation

Clone the repository:

git clone https://github.com/Omer-Girginer/To-do-app

Install dependencies:

npm install

Run the application:

npm start

## Purpose

The purpose of this project is to practice React fundamentals such as components, state management, and event handling.
